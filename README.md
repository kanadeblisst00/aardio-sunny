## sunny介绍

这类似于Python的mitmproxy，可以自定义脚本修改监听到的数据。相比于mitmproxy优点更多，比如：

- 支持更多的数据类型，比如tcp、udp和websockets等，
- 可以不使用代理使用驱动来监听某个进程的数据
- 封装成了dll，任何语言都可以很方便的调用它

官方地址: `https://esunny.vip/`

github开源地址: `https://github.com/qtgolang/SunnyNet`
